🎮 Tetria Game (Tetris Inspired Game)

 📖 Project Description

Tetria Game is a classic block puzzle game inspired by Tetris. The main objective of the game is to arrange falling blocks in such a way that they form complete horizontal lines. Once a line is completed, it gets cleared and the player earns points. The game continues until there is no space left to spawn a new piece.

---

 ✨ Features

- Classic Tetris-style gameplay  
- Random tetromino generation  
- Smooth collision detection  
- Piece rotation and movement  
- Automatic line clearing  
- Score tracking system  
- Game Over detection  
- Responsive game rendering  

---

 🛠️ Technologies Used

- HTML5  
- CSS3  
- JavaScript  

---

 🎮 Controls

| Key | Action |
|-----|--------|
| ← | Move Left |
| → | Move Right |
| ↓ | Move Down |
| ↑ | Rotate Piece |

---

📜 Game Rules

- Blocks fall from the top of the board.  
- Move and rotate the blocks to fit them together.  
- Complete a full horizontal row to clear it.  
- Each cleared row increases your score.  
- Game ends when a new piece cannot be placed.
