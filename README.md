Tetria Game
Overview

Tetria Game is a classic block puzzle game inspired by Tetris. The objective is to arrange falling blocks to complete horizontal lines. Completed lines are cleared, and the player earns points. The game continues until there is no space to spawn a new piece.


Features
Classic Tetris-style gameplay
Random tetromino generation
Collision detection
Piece rotation and movement
Automatic line clearing
Score tracking
Game Over detection
Responsive game rendering
Technologies Used
HTML5
CSS3
JavaScript
Controls
Key	Action
←	Move Left
→	Move Right
↓	Move Down
↑	Rotate Piece
Game Rules
Blocks fall from the top of the board.
Move and rotate the blocks to fit them together.
Complete a full horizontal row to clear it.
Each cleared row increases your score.
The game ends when a new piece cannot be placed at the top of the board.
